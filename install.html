Install TAGUETTE

Taguette is released under BSD-3-Clause (read the license file here), © Rémi Rampin. You can read about the team behind Taguette on our About page.

Installers

Installers are available from MacOS and Windows from our latest release page. Those include Python, Taguette, and Calibre, so you don't need to install anything else.

Pre-Requisites

To use Taguette right now, you first need to install Python 3 and Calibre, the open source e-book management software. Taguette uses a part of Calibre to convert uploaded documents into HTML, allowing you to highlight and tag parts of them.

Installation from terminal

Once you've installed the pre-reqs, you can install Taguette! You can quickly install Taguette with the following: pip install taguette

After which you can simply run taguette in the terminal to get it going. You'll see the command line will still be running. This is ok! Don't worry about the terminal but do leave it running. If you kill the terminal window, you'll kill Taguette!

In your web browser, navigate to localhost:8000 to begin working on your projects. Learn more about uploading and tagging materials at our getting started page!

Development set-up

You can follow these steps to use Taguette from source:

  1. Clone this git repository: git clone https://gitlab.com/remram44/taguette.git
  2. Navigate on the command line to the repository you've just cloned locally, using the cd command. To get help using cd, use this tutorial.
  3. To install the dependencies of Taguette, enter the command pip install -e . in the terminal.
    • We recommend you run this inside of a virtualenv or pipenv if possible.
  4. Once the dependencies have been installed, enter the command taguette in the terminal and you'll see it will stay running. This is ok! Don't worry about the terminal but do leave it running. In your web browser, navigate to localhost:8000 to begin working on your projects!

Learn about using Taguette at our getting started page!
